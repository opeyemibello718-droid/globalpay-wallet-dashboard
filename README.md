# GlobalPay Wallet Dashboard

## Overview
GlobalPay Wallet Dashboard is a web application designed to manage digital wallet transactions and monitor user accounts efficiently.

## Features
- **User Authentication**: Secure login and registration for users.
- **Transaction Management**: View, add, and manage transactions in real-time.
- **Account Monitoring**: Keep track of account balances and transaction history.
- **User-Friendly Interface**: Intuitive design for an enhanced user experience.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/opeyemibello718-droid/globalpay-wallet-dashboard.git
   ```
2. Navigate to the project directory:
   ```bash
   cd globalpay-wallet-dashboard
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
1. Start the application:
   ```bash
   npm start
   ```
2. Access the application via `http://localhost:3000` in your web browser.
